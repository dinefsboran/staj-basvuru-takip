# Staj Başvuru Takip Sistemi

Bu proje, ReactJS kullanılarak geliştirilmiş basit bir staj başvuru takip uygulamasıdır. Kullanıcılar başvurdukları firmaları, pozisyonları, başvuru tarihlerini ve başvuru durumlarını tek bir ekrandan takip edebilir.

## Proje Amacı

Bu uygulamanın amacı, staj başvurularını düzenli bir şekilde kaydetmek ve takip etmektir. Proje kapsamında modern JavaScript kütüphanesi olan ReactJS kullanılmıştır. Uygulama; ekleme, listeleme, güncelleme ve silme işlemlerini içermektedir.

## Kullanılan Teknolojiler

- ReactJS
- Vite
- JavaScript
- HTML
- CSS
- LocalStorage
- Netlify
- GitHub

## Proje Özellikleri

- Yeni staj başvurusu ekleme
- Eklenen başvuruları listeleme
- Başvuru bilgilerini güncelleme
- Başvuruları silme
- Başvuru durumunu seçme
- Verileri tarayıcıda LocalStorage ile saklama
- Responsive tasarım

## Başvuru Bilgileri

Her başvuru kaydında aşağıdaki bilgiler tutulur:

- Firma adı
- Pozisyon
- Başvuru tarihi
- Başvuru durumu

## Başvuru Durumları

Uygulamada kullanılabilen başvuru durumları:

- Beklemede
- Mülakat
- Kabul
- Red
